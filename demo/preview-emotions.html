🐥 Savy 表情设计预览

基于玻尿酸鸭设计规范，用于课程节点内表达情绪状态

📐 设计规范

  • 统一角色：玻尿酸鸭（Lalafanfan Duck）— 黄色圆头、粉色腮红、橙色鸭嘴、金色圆框眼镜、粉色项圈+金色铃铛
  • 表情差异：通过眼睛形态（眯眼/大眼/泪眼）、嘴巴形态（大笑/微笑/嘟嘴/抿嘴）、手臂动作（举手/挥手/垂下/托腮）、装饰元素（彩纸/星星/乌云/问号）来区分情绪
  • 参考原版：跳跃开心、困打哈欠、睁大眼睛、抱臂思考等经典玻尿酸鸭表情
  • 文件命名：savy-{emotion}.svg，尺寸 120×120（celebrate 为 120×130）
🎉 Celebrate

庆祝

跳跃腾空 + 眯眼大笑 + 双手举起欢呼
周围撒花彩纸 + 小星星
用于：答对题目、完成课程、达成成就
😊 Happy

开心

睁大亮晶晶眼睛 + 微笑鸭嘴
右手挥手打招呼 + 动态线
用于：欢迎页、正确反馈、鼓励提示
😢 Sad

伤心

泪汪汪大眼 + 八字委屈眉 + 嘟嘴
双手垂下 + 头顶小乌云下雨
用于：答错题目、连续失败、超时未完成
🤔 Thinking

思考

眼镜推到头顶 + 一眯一睁 + 抿嘴
右手托腮 + 思考泡泡问号 + 小齿轮
用于：等待作答、提示思考、加载中
